# BRAIAIN Standard — Test Infrastructure

Complete test battery for scoring AI models 400–1600.

## Quick start

```bash
pip install anthropic openai google-generativeai

# Set environment variables (NEVER pass keys on CLI)
export BRAIAIN_API_KEY=sk-...
export BRAIAIN_JUDGE_KEY=sk-...
export BRAIAIN_SEED_SALT=$(openssl rand -hex 16)   # Keep secret. Do NOT commit.
export BRAIAIN_QUARTER=2026Q2                       # Update each quarter

# Step 1: Generate novel questions for this quarter
python generators/math_gen.py       # → questions/novel_math.json
python generators/bug_inject.py     # → questions/novel_coding.json
python generators/logic_gen.py      # → questions/novel_reasoning.json
python generators/science_gen.py    # → questions/novel_science.json

# Step 2: Run the full battery against a model
python run.py --model gpt-4o --provider openai

# Step 3: Run AI judge on Tier 2 answers
python judge.py \
  --result output/results/gpt-4o-YYYYMMDD-HHMM.json \
  --judge-provider anthropic

# Step 4: Human expert reviews Tier 3 answers
# Edit output/human/gpt-4o-YYYYMMDD-tier3.json manually

# Step 5: Publish final scores
python update_models.py \
  --result output/results/gpt-4o-YYYYMMDD-HHMM-judged.json \
  --human-review output/human/gpt-4o-YYYYMMDD-tier3.json \
  --models-json ../models.json
```

## Scoring system

```
8 Dimensions → 2 Sections → 1 Total

Analytical (200–800) = 200 + (R×0.35 + M×0.40 + S×0.25) × 600
Technical  (200–800) = 200 + (C×0.40 + X×0.30 + E×0.15 + V×0.15) × 600
Total (400–1600) = Analytical + Technical

Multimodal: recorded but not in formula. Non-vision models get null (not mean).
```

## Three-tier judging

| Tier | Who | What | ~% of questions |
|------|-----|------|-----------------|
| 1 | Automated | Unit tests, exact match, list comparison, word count | 55% |
| 2 | AI judge panel | Rubric scoring via structured prompt | 35% |
| 3 | Human expert | Proofs, complex explanation, disputes | 10% |

**Judge selection:** No model judges its own provider family. Hard enforced — judge.py exits with error code 1 if violated.

## Security and integrity

- **API keys**: Environment variables only. CLI flags removed.
- **Seed salt**: Seeds derived from BRAIAIN_QUARTER + BRAIAIN_SEED_SALT. Never commit the salt.
- **Immutability hash**: SHA256 of model ID + question IDs + prompts + responses.
- **Self-judging**: Hard sys.exit(1) if judge matches model family.

## Contamination detection

Bootstrap CI (n=10,000) on fixed-vs-novel gap per dimension.
Flag if 95% CI lower bound > 0 AND gap > 0.10.
Warn if raw gap > 0.20 but CI includes zero.

## Before first run

1. Set BRAIAIN_SEED_SALT and keep it secret
2. Dry run: `python run.py --model gpt-4o --provider openai --dry-run`
3. Human expert reviews 100% of calibration answers
4. Run real test on GPT-4o as first published score

**Estimated time to first legitimate published score: 8–10 weeks.**
